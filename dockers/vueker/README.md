# vueyoutube

> My youtube Project

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report
```

Oh and don't forget to go to server folder with `cd server/` and run `node index.js` for ElasticSearch ..
